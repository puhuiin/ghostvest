技术对比

HMAC-SHA256 确定性派生 vs ZK 证明 vs 混币器，在兼容性、速度和复杂度之间的最优平衡。

维度GhostVest (HMAC)ZK-basedMixer
钱包兼容性100% (任何 ed25519)需特定钱包需特定合约
Gas 成本可忽略高（证明验证）中等
延迟< 100ms5-30 秒不定
确定性同输入 → 同地址因实现而异非确定性
链上足迹单笔转账复杂 calldata多跳
可重新派生✓ nonce + pubkey✗✗

快速开始

4 步启动 GhostVest 本地开发环境。

克隆仓库

git clone https://github.com/puhuiin/ghostvest.git && cd ghostvest

启动后端

cd backend && pip install -r requirements.txt
python -m uvicorn app.main:app --port 8000

启动前端

cd frontend && npm install && npm run dev

打开浏览器

访问 http://localhost:3000/claim，连接 Phantom 钱包，点击 Claim 体验全流程。
